Amazon Sales Analysis using Python and Excel
Project Overview

This project analyzes Amazon sales transaction data to understand business performance, identify top-performing products and categories, and provide actionable business recommendations.

Technologies Used
Python
Pandas
Matplotlib
Google Colab
Microsoft Excel
Dataset Features
Sales Transactions
Product Categories
Customer Information
Payment Methods
State-wise Sales Data
Analysis Performed
KPI Analysis
Sales by State
Sales by Category
Top Products
Top Customers
Monthly Sales Trend
Payment Analysis
Interactive Excel Dashboard
Visualizations
Bar Chart
Donut Chart
Line Chart
Horizontal Bar Chart
Business Recommendations
Focus on high-performing categories such as Electronics.
Improve sales in underperforming states.
Provide offers on popular payment methods.
Dataset Limitation

The provided dataset does not contain:

Profit column
Sub-Category column

Therefore, profit-based and sub-category analyses could not be performed.
