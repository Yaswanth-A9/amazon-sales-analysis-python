# Amazon Sales Analysis using Python and Excel

## 📌 Project Overview
This project analyzes Amazon sales transaction data to understand business performance, identify top-performing products and categories, and provide actionable business recommendations.

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab
- Microsoft Excel

---

## 📂 Dataset Features
- Sales Transactions
- Product Categories
- Customer Information
- Payment Methods
- State-wise Sales Data

---

## 📊 Analysis Performed
- KPI Analysis
- Sales by State
- Sales by Category
- Top Products Analysis
- Top Customers Analysis
- Monthly Sales Trend
- Payment Analysis
- Interactive Excel Dashboard

---

## 📈 Visualizations
- Bar Chart – Sales by State
- Donut Chart – Category-wise Sales
- Line Chart – Monthly Sales Trend
- Horizontal Bar Chart – Top 10 Products by Sales

---

## 💡 Business Recommendations
1. Focus on high-performing categories such as Electronics.
2. Improve sales in underperforming states through targeted marketing campaigns.
3. Provide offers and cashback on popular payment methods to increase customer retention and sales.

---

## ⚠️ Dataset Limitations
The provided dataset does not contain:
- Profit column
- Sub-Category column

Therefore:
- Profit-based analysis could not be performed.
- Sub-category analysis could not be performed.
